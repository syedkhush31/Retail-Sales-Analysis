# Retail Sales Analysis (Excel + Python)

## Project Overview
This project analyzes a large retail sales dataset to understand sales trends, category performance, regional distribution, and product-level insights. The goal is to use data to support business decision-making rather than intuition.

## Tools Used
- Python
- pandas
- matplotlib
- Excel
- Git & GitHub

## Dataset
The dataset contains retail sales records with information such as:
- Order Date
- Category and Sub-Category
- Region
- Sales values

The dataset spans multiple years, allowing time-based trend analysis.

## Analysis Performed
- Cleaned and prepared raw sales data
- Converted and validated date fields
- Performed monthly sales trend analysis
- Analyzed sales by category and sub-category
- Analyzed regional sales performance
- Created clear and readable visualizations for trends and comparisons

## Key Insights
- Sales show strong month-to-month fluctuations, indicating seasonality.
- Technology is the highest revenue-generating category.
- Sales performance varies significantly across regions.
- A small number of sub-categories contribute a large portion of total sales, highlighting key focus areas for inventory and marketing strategies.

## Conclusion
This analysis demonstrates how structured data analysis can uncover meaningful business insights from large datasets and support data-driven decision-making.
